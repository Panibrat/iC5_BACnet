## React + Redux + Jest + NodeJS + MongoDB

##### to start MongoDB Server use windows cmd:
```
"C:\Program Files\MongoDB\Server\3.4\bin\mongod.exe" --dbpath c:\dbs
```
##### to create AV + BV points in MongoDB:
```
npm run create-db
```
##### to start webpack:
```
webpack
```
##### to start server open +1 cmd:
```
npm start
```
